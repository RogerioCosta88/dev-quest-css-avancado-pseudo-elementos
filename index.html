Pseudo Elementos

Servem para estilizar partes especificas de elementos, como: estilizar a primeira letra ou primeira linha de um <p>.

Anatomia

seletor::pseudo-elemento {
                propriedade: valor;
}

First Letter

É usado para alterar a primeira letra de um elemento. Podemos usar classes e id.

First Line

Usamos o first-line para estilizar a primeira linha de um elemento. Lorem ipsum dolor sit amet consectetur adipisicing elit. Nam architecto laborum unde repudiandae sint exercitationem dolorum fugiat harum at, beatae, vel voluptate dolores ex suscipit molestiae quod, inventore quas deleniti.

Marker

Usamos o marker para estilizar os marcadores de uma lista

  • item 1
  • item 2
  • item 3
  • item 4

Before

É usado para inserir um conteúdo antes do elemento que estamos estilizando. Podendo ser uma imagem ou um texto que queremos colocar antes de um ou mais <h1>, por exemplo.

After

Já o pseudo-elemento after, vai incluir o conteúdo após o elemento estilizado. Este conteúdo também pode ser incluido em vários elementos ao mesmo tempo.
